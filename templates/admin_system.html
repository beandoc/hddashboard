Hospitalization Events Collected
What counts as an event? Each monthly record where any of the following is set contributes one training example:
  • Hospitalization This Month checkbox is checked
  • Hospitalization Diagnosis field has text (e.g. "CHF exacerbation")
  • Hospitalization ICD Code field has text (e.g. "I50.9")
How hospitalization diagnosis affects training: The diagnosis text is not used as a model feature — it only signals that a hospitalization occurred. Filling it in (alongside the checkbox) reinforces the positive label and helps the model learn which clinical patterns precede admission. More accurately-labelled events → better-calibrated risk scores.
